# GSSG

为个人网站项目，网站名称为 __GSSG__

分为前端后端  
文件目录

1. 前端 frontEnd-antdPro  
2. 后端 backEnd  

## 前端

框架主要用到 __React__ __TypeScript__ __antdPro__

### 快速开始

```bash
# 安装依赖包
npm install

# 启动项目
npm run start
```

### 打包

```bash
npm run build
```

## 后端

框架主要用到 __Node.js__ __express__

使用 __ORM__ 工具[Sequelize](https://sequelize.org/ "Sequelize官网")

### 快速开始

```bash
# 安装依赖包
npm install

# 启动项目
npm run dev
```
